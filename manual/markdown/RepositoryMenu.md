Using the Repository Menu

## Summary (lead paragraphs)
The Repository Menu controls options for the Repository and can be accessed either by clicking on the Repository icon at the top-right corner of a Templates or Script, or by right-clicking on a Template or Script and selecting Repository from the menu. Note that 'Template' or 'Script' should appear in the status bar when editing a Template or Script and see Using Edit As for more information. Note that the Repository Menu is also available in the Repository Dialog in a slightly different form by clicking the down-arrow to the right of the Uninstall button for an installed Template or Script.
